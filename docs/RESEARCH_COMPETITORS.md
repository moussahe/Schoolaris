## Resume Executif

Le marche EdTech francais represente **1,6 milliard d'euros** en 2023 avec une croissance annuelle de **11%**. Le secteur compte plus de **540 startups** et **15 000 employes**. Le soutien scolaire prive concerne plus d'**un million d'eleves** pour un marche evalue a **2 milliards d'euros**.

**Constat cle** : Le marche est domine par des acteurs matures (Kartable, SchoolMouv, Maxicours, Les Bons Profs) qui proposent des offres similaires. Les principales opportunites resident dans :

- L'integration de l'IA generative pour la personnalisation
- La gamification avancee (sous-exploitee)
- Le segment primaire (CP-CM2) moins concurrentiel
- L'accompagnement parental renforce

## Analyse du Marche EdTech Francais

### Taille et Croissance

| Indicateur                  | Valeur        |
| --------------------------- | ------------- |
| CA EdTech France 2023       | 1,6 Md EUR    |
| Croissance annuelle moyenne | 11%           |
| Nombre de startups EdTech   | 540+          |
| Employes du secteur         | 15 000        |
| Marche mondial EdTech 2024  | 172,71 Md USD |
| Projection mondiale 2031    | 422,95 Md USD |

### Tendances 2024-2025

1. **Intelligence Artificielle** : Le marche de l'IA dans l'education passe de 5,88 Md USD (2024) a 32,27 Md USD (2030) - croissance de +41% en 2025
2. **Formats hybrides** : 37% des formations sont en blended learning
3. **Personnalisation** : Demande croissante pour des parcours adaptatifs
4. **Gamification** : Reste sous-exploitee dans l'education formelle
5. **Consolidation** : Les financements se concentrent sur les structures matures

### Defis du Secteur

- Levees de fonds en baisse (3 Md USD en 2024 vs 17 Md USD en 2021)
- Fragmentation du marche francais
- Difficulte a penetrer le secteur public
- Imperatif de rentabilite rapide

### 1. Kartable

**Site** : [kartable.fr](https://www.kartable.fr)

**Presentation**

- Fonde en 2012 a Bordeaux/Paris
- Fondateurs : Sarah Besnainou et Julien Cohen-Solal (HEC Paris)

**Modele Economique** : Freemium avec abonnement

**Grille Tarifaire**

| Contenus      | 14,99 EUR/mois | -              | 9,99 EUR/mois | 7,99 EUR/mois  |

### 2. SchoolMouv

**Grille Tarifaire**

### 4. OpenClassrooms

**Site** : [openclassrooms.com](https://openclassrooms.com)

**Presentation**

- Leader de la formation professionnelle en ligne
- Diplomes reconnus par l'Etat (RNCP, Bac+2 a Bac+5)
- Modele de mentorat individuel

**Modele Economique** : Freemium + abonnement premium

**Grille Tarifaire**

| Formule             | Prix                                      |
| ------------------- | ----------------------------------------- |
| Gratuit             | 5 videos/semaine, forums, tests           |
| Premium             | 20 EUR/mois (acces illimite, certificats) |
| Premium Plus        | 300 EUR/mois (diplomes, mentorat)         |
| Parcours metiers    | 300-500 EUR/mois (9 mois en moyenne)      |
| Starter Entreprises | 360 EUR/licence/an                        |

**Cible** : Adultes en reconversion, etudiants superieurs, entreprises

**Forces**

- Diplomes reconnus par l'Etat
- Mentorat individuel
- Eligible CPF
- Large catalogue de cours gratuits
- Forte notoriete

**Faiblesses**

- Pas cible scolaire (college/lycee)
- Prix eleves pour parcours diplomanmts
- Cout final depend du rythme de l'apprenant

**Features Uniques**

- Diplomes RNCP
- Mentorat one-to-one
- Financement CPF/Pole Emploi

**Note** : OpenClassrooms n'est pas un concurrent direct pour le soutien scolaire CP-Terminale mais represente un modele interessant pour l'IA et le mentorat.

**Site** : [superprof.fr](https://www.superprof.fr)

**Presentation**

- Marketplace de mise en relation profs/eleves
- Note Trustpilot : 4,3/5 (9 291 avis)
- Modele "Uber" de l'education

**Modele Economique** : Commission sur mise en relation

**Grille Tarifaire**

| Element                           | Prix                                   |
| --------------------------------- | -------------------------------------- |
| Pass Eleve                        | 39 EUR/mois (sans engagement)          |
| Commission profs (via plateforme) | 10% apres 1er cours                    |
| Inscription profs                 | Gratuit                                |
| Tarif moyen cours                 | 18,90-50 EUR/heure selon matiere/ville |

**Cible** : Tous ages, tous niveaux, parents

**Forces**

- Large choix de profs (toutes matieres)
- Flexibilite totale (horaires, lieu)
- Premier cours souvent gratuit
- Pas de commission pour les profs (hors plateforme)
- Avis verifies sur les profs

**Faiblesses**

- Pas de contenus propres
- Qualite variable des profs
- 39 EUR/mois juste pour la mise en relation
- Pas de suivi pedagogique structure

**Features Uniques**

- Marketplace pure
- Profs pour toutes les disciplines (musique, sport, langues...)
- 100% du tarif au prof (hors paiement plateforme)

## Opportunites Non Exploitees

### 1. Intelligence Artificielle Generative

**Constat** : Aucun acteur majeur n'a integre de maniere significative l'IA generative (type ChatGPT/Claude) pour :

- Tuteur IA personnalise 24/7
- Generation d'exercices adaptatifs
- Correction intelligente avec explications
- Aide aux devoirs en temps reel

**Exemple emergent** : Rakoono (startup 2024) propose un chatbot IA pour collegiens/lyceens mais reste confidentiel.

### 2. Gamification Avancee

**Constat** : La gamification reste "basique" (points, badges) sur toutes les plateformes.

**Opportunite** :

- Jeux educatifs immersifs (modele PowerZ)
- Systeme de quetes et progression narrative
- Competition saine entre eleves (classements, defis)
- Recompenses motivantes

### 3. Segment Primaire (CP-CM2)

**Constat** :

- Les Bons Profs ne couvre pas le primaire
- Les autres ont une offre primaire moins developpee que college/lycee
- 20% des eleves ne maitrisent pas les fondamentaux en fin de primaire

**Opportunite** : Se specialiser sur le primaire avec une approche ludique et adaptee aux jeunes enfants.

### 4. Accompagnement Parental Renforce

**Constat** : Les parents ont un "compte parent" pour suivre la progression mais peu d'outils pour accompagner activement.

**Opportunite** :

- Conseils personnalises pour les parents
- Alertes proactives sur les difficultes
- Ressources pour aider son enfant
- Communication directe avec l'equipe pedagogique

### 5. Integration Ecole/Enseignants

**Constat** : Les plateformes sont centrees B2C, peu d'integration avec le systeme scolaire.

**Opportunite** :

- Partenariats avec etablissements
- Outils pour enseignants (suivi classe)
- Alignement programme en temps reel
- Rapports pour conferences parents/profs

### 6. Accessibilite et Inclusion

**Constat** : Peu d'adaptation pour eleves a besoins specifiques (dys, TDAH, handicap).

**Opportunite** :

- Contenus adaptes dys
- Mode concentration pour TDAH
- Accessibilite visuelle/auditive
- Parcours personnalises selon profil

### 7. Apprentissage par Projets

**Constat** : Toutes les plateformes sont basees sur le modele cours/exercices traditionnel.

**Opportunite** :

- Projets concrets et transversaux
- Application des connaissances
- Portfolio de realisations
- Collaboration entre eleves

## Recommandations Strategiques pour Schoolaris

### Positionnement Recommande

**"La plateforme EdTech IA-first pour les scolaires francais"**

Schoolaris doit se differencier en etant le premier acteur a integrer veritablement l'IA generative au coeur de l'experience d'apprentissage, tout en maintenant une approche humaine et ludique.

### Avantages Competitifs a Developper

#### 1. Assistant IA "Claude" - Differenciateur Majeur

- **Tuteur IA 24/7** : Aide aux devoirs instantanee avec explications personnalisees
- **Generation d'exercices** : Exercices adaptes au niveau et aux lacunes de l'eleve
- **Correction intelligente** : Feedback detaille avec pistes d'amelioration
- **Mode Socratique** : L'IA guide vers la solution sans donner la reponse

**Positionnement prix** : Inclure l'IA dans toutes les formules (pas en option premium)

#### 2. Gamification Immersive

- **Univers narratif** : Histoire/quete qui se deroule avec la progression
- **Avatar personnalisable** : Evolution visuelle avec les accomplissements
- **Defis quotidiens** : Micro-objectifs pour maintenir l'engagement
- **Classements bienveillants** : Competition positive sans stigmatisation
- **Recompenses tangibles** : Partenariats pour recompenses reelles (livres, etc.)

#### 3. Focus Primaire (CP-CM2)

- **UX enfant-first** : Interface coloree, intuitive, sans texte complexe
- **Personnages mascotte** : Guides pedagogiques attachants
- **Sessions courtes** : 15-20 minutes adaptees a l'attention des enfants
- **Validation parentale** : Parents impliques dans le parcours

#### 4. Accompagnement Parental Actif

- **Dashboard parent avance** : Insights actionables sur les progres
- **Alertes intelligentes** : Notifications proactives sur les difficultes
- **Conseils personnalises** : "Cette semaine, travaillez les tables de multiplication"
- **Ressources parent** : Comment aider son enfant selon son profil
