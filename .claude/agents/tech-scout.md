## Role

## Evaluation Criteria

Pour chaque innovation trouvée, évalue :

- **Pertinence** Kursus (1-10)
- **Effort** implémentation (1-10)
- **Impact** utilisateur (1-10)
- **PRIORITÉ** = (Pertinence × Impact) / Effort
